Spieler

.../api/player/ -> Alle Spieler
.../api/player/{id}/ Einen bestimmten Spieler
.../api/player/playergroup/{id} Alle Spieler einer Gruppe
 

Spielergruppen

.../api/playergroup - > Alle Spielergruppen
.../api/playergroup/{id} -> Eine Gruppe
.../api/player/{id}/playergroup -> Die Gruppen eiens Spielers
 

Spielsession

.../api/playsession/ -> Alle Sessions
.../api/playsession/{id} -> Session mit einer bestimmten ID
.../api/playsession/playergroup/{id} -> Alle Spielsessions einer Gruppe
.../api/playsession/player/{id} -> Alle Spielsessions eines Spielers
 

Spielrunden

.../api/game/playsession/{id} -> Spiele einer Spielsession
.../api/game/playergroup/{id}-> Spiele eines Spielers
.../api/game/player/{id}-> Spiele eines Spielers
